# Cake Baking Simulation Application 🎂

## 📌 Project Description
This project is a desktop application that simulates the cake baking process step by step using timers and progress bars.

## 🧠 Technologies Used
- C#  
- Windows Forms  
- .NET Framework  

## 📊 Features
- Step-by-step baking simulation  
- Timer-based process control  
- Progress bar visualization  
- User interaction with start button  

## 🎯 Goal
To simulate a real-life process (cake baking) using programming concepts like timers and UI components.

## 🚀 Status
Completed as a C# Windows Forms project.
